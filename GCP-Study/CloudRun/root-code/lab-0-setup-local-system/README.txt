|---------------------------------------------------------------------------------------------------------------|
|Install Python: (Optional)                                                                                     |
|---------------------------------------------------------------------------------------------------------------|
|Mac (With Homebrew): brew install python                                                                       |
|Mac (Without Homebrew): https://www.python.org/downloads/                                                      |
|Windows: https://www.python.org/downloads/release/python-3114/                                                 |
|                                                                                                               |
|---------------------------------------------------------------------------------------------------------------|
|Install gCloud CLI: (Optional)                                                                                 |
|---------------------------------------------------------------------------------------------------------------|
|Mac (With Homebrew): brew install --cask google-cloud-sdk                                                      |
|Mac (Without Homebrew): https://cloud.google.com/sdk/docs/install#mac                                          |           
|Windows: https://cloud.google.com/sdk/docs/install#windows                                                     |
|                                                                                                               |
|---------------------------------------------------------------------------------------------------------------|
|Install git CLI:                                                                                               |
|---------------------------------------------------------------------------------------------------------------|
|Mac (With Homebrew): brew install git                                                                          |
|Mac (Without Homebrew): https://git-scm.com/book/en/v2/Getting-Started-Installing-Git                          |
|Windows: https://git-scm.com/book/en/v2/Getting-Started-Installing-Git                                         |
|                                                                                                               |
|---------------------------------------------------------------------------------------------------------------|
|Install git credential manager:                                                                                |
|---------------------------------------------------------------------------------------------------------------|
|Mac (With Homebrew): brew install --cask git-credential-manager                                                |
|Mac (Without Homebrew): https://github.com/git-ecosystem/git-credential-manager/blob/release/docs/install.md   |
|Windows: https://github.com/git-ecosystem/git-credential-manager/blob/release/docs/install.md                  |
|                                                                                                               |
|---------------------------------------------------------------------------------------------------------------|
|Install Visual Studio Code:                                                                                    |
|---------------------------------------------------------------------------------------------------------------|
|Mac (With Homebrew): brew install --cask visual-studio-code                                                    |
|Mac (Without Homebrew): https://code.visualstudio.com/download                                                 |
|Windows: https://code.visualstudio.com/download                                                                |
|                                                                                                               |
|---------------------------------------------------------------------------------------------------------------|
|Install Visual Studio Code REST Client:                                                                        |
|---------------------------------------------------------------------------------------------------------------|
|https://marketplace.visualstudio.com/items?itemName=humao.rest-client                                          |
|                                                                                                               |
|---------------------------------------------------------------------------------------------------------------|
|Verify everything is installed correctly                                                                       |
|---------------------------------------------------------------------------------------------------------------|
|                                                                                                               |
|python --version                                                                                               |
|pip --version                                                                                                  |

|python3 --version                                                                                              |
|pip3 --version                                                                                                 |
|                                                                                                               |
|gcloud --version                                                                                               |
|git --version                                                                                                  |
